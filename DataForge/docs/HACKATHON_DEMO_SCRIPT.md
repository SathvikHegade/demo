# DataForge Sentinel - 5 Minute Demo Script

## 0:00-0:40 Problem and Impact
- "Most ML failures start with bad data. DataForge Sentinel catches quality risks before model training."
- "We analyze bias, noise, duplication, and class imbalance, then output an AI-grounded remediation plan."

## 0:40-1:30 Live Dataset Analysis
- Open the app and upload a noisy sample file.
- Add `target` and sensitive columns (for fairness checks).
- Run analysis and call out processing time and deterministic output.

## 1:30-2:40 Judge-Focused Insight Walkthrough
- Highlight global quality score and deployment readiness.
- Show 4 pillar scores:
  - Bias
  - Noise
  - Duplication
  - Imbalance
- Open AI executive summary and show top action list with effort tags.

## 2:40-3:40 Innovation Moment: What-if Simulation
- Use the "What-if Improvement Simulator" sliders.
- Show before vs after score delta in real time.
- Explain this helps teams prioritize highest-ROI cleaning actions.

## 3:40-4:30 Technical Depth and Integration
- Mention architecture:
  - Modular analyzer engine
  - FastAPI + CLI + HTML/JSON reports
  - Grounded AI narrative generated from metrics
- Show `/docs` and one report export endpoint.

## 4:30-5:00 Closing: Real-World Fit
- "This is not just visualization; it's an operational quality gate."
- "Teams can automate this before every training run."
- End with final score improvement and readiness verdict.

## Rubric Mapping
- Real-World Impact: quality gate before training.
- Technical Depth: modular analyzers, scoring contract, API/CLI.
